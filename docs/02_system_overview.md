## 連携システム

| システム | 役割 |
|----------|------|
| Modrinth API | プロジェクト情報・バージョン情報・ファイルの取得元 |
| ローカルファイルシステム | 設定ファイルの読み込み、ダウンロードファイルの保存先 |

## 利用者ができること

### 1. 管理対象の確認

- 現在管理している MOD/shader/リソースパックの一覧を表示する
- 各プロジェクトのインストール状態を確認する

### 2. 更新確認

- Modrinth API に問い合わせて、新しいバージョンがあるか確認する
- 更新があるプロジェクトを一覧表示する

### 3. インストール・更新

- 新しいバージョンをダウンロードする
- ハッシュ検証で整合性を確認する
- 適切なディレクトリに配置する

### 4. 管理対象の追加・削除

- 新しいプロジェクトを管理対象に追加する
- 不要になったプロジェクトを管理対象から削除する

## スコープ

### このプロジェクトでやること

- TOML ファイルで管理対象を定義
- Modrinth API からプロジェクト情報を取得
- 更新確認とダウンロード
- CLI インターフェースの提供

### このプロジェクトでやらないこと

- Minecraft のインスタンス管理
- Minecraft の起動（ランチャー機能）
- MOD 間の依存関係解決（将来の拡張として検討）
- CurseForge など他のプラットフォームへの対応
- GUI（TUI は将来の拡張として検討）
